- 擲筊結果：
    - 為隨機出現，每按一次再問一題，皆需重新計算，後端不用紀錄前一次的擲出結果。
    - 會有３種結果：
        - 聖筊(一正一反) → 視為神明允許詢問。
        - 笑杯(2反，平面朝上) → 視為神明未決定（笑而不答），或說得不清楚。

### 2.2.2 擲筊流程

1. **開始擲筊**
    - 點擊場景中的「擲筊」熱點。
    - 進入【第一關：擲筊】。
2. **第一關：擲筊**
    - 上方變成「結束擲筊」熱點，其他原頁面熱點皆不能點擊，避免重複觸發。
    - 畫面中央出現「3D 筊杯」，3D 筊杯會跟著使用者滑鼠移動。
    - 使用者在指定區域內大幅度拖曳（Mobile 設備以點擊替代拖曳），觸發擲筊動作。
    - 畫面顯示筊杯拋起與落下的動畫。
    - 系統隨機計算結果（後端不紀錄前次結果，每次皆為獨立機率計算）。
    - 進入【第二關：顯示結果與重置】。
3. **第二關：顯示結果與重置**
    - 筊杯落下後，隨機顯示本次結果為「聖筊 / 笑筊 / 陰筊」等。
    - 不管得到什麼結果，皆在 3D 筊杯中央顯示「再問一題」文字按鈕（Hover 過去時，文字會放大）。
    - 使用者點擊「再問一題」。
    - 筊杯回到擲出前的初始樣式，退回【第一關：擲筊】等待下一次操作。
4. **隨時結束機制**
    - 在上述任何流程中，使用者皆可隨時點擊上方的「結束擲筊」熱點。
    - 結束擲筊流程，畫面返回凝輝殿主場景。

- 抽屜打開後，卡面會迷你狀態，逐漸放大到中心位置，3D籤筒會逐漸向左移動到「第一關」初始位置，進入最後一關。
- 卡片彈出過程，所需縮圖可以用相對應詩籤庫第一張替代，抵達對應位置是使用者抽中得「第N首」圖片即可。
- 飛卡動畫中，卡片 scale 從 `0.05` 漸變到 `4` 。(供參考)

**放大縮小 邏輯**

- 級距：
    - 每次點一次 + / -：`0.35`

    - 最小值：`1.2`
- **初始值：**
    - 進入這關的飛卡動畫中，卡片 scale 從 `0.05` 漸變到 `4` 。(供參考)
    - 動畫結束後，`sixthLayerCardScaleRef.current` 會被設成 `4`（操作起點）。

### 2.3.6 求籤流程

1. **開始求籤**
    - 點擊場景中的「求籤」熱點。
    - 選擇類別後，進入【第一關：擲筊請示】。
2. **第一關：擲筊請示**
    - 原位顯示「3D 籤筒」，上方變成「結束求籤」熱點。
    - 畫面中央出現「3D 筊杯」。
    - 使用者點擊擲筊。
    - 判斷結果：
        - 獲取聖杯 -> 進入【第二關：確認抽籤類別】。
        - 失敗超過 5 次 -> 提示「宜明日再詢問」文字，結束流程(理論上可以無限重擲，如有流量問題可限制5次。)。
3. **第二關：選擇類別**
    - 原位顯示「3D 籤筒」，上方變成「問人事」、「問修道」熱點。
    - 使用者點擊類別。
    - 系統依據問事類別（問修道、問人事），選擇後，鎖定卡片庫和可抽籤之數量，後面卡面為對應卡面庫。
    - 選擇類別後，進入【第三關：抽籤】。
4. **第三關：抽籤**
    - 籤筒拉進至中間位置。
    - 畫面顯示「籤筒」和「點擊籤筒」提示。
    - 使用者滑鼠移到籤筒上半部詩籤們的區域時，籤會在滑鼠左右移動。
    - 使用者點擊籤筒。
    - 系統依據問事類別（問修道60隻、問人事49隻）給出可出現的數字。
    - 播放抽籤動畫，進入【第四關：確認詩籤】。
5. **第四關：確認詩籤**
    - 播放抽籤動畫，抽出一支「3D 籤」於畫面右方。
        - 「3D 籤筒」移回原位，上方變成「結束求籤」熱點。
        - 中間出現「3D筊」，筊中間顯示「確認詩籤」供使用者點擊。
        - 右側浮現「3D 籤」（顯示「第 N 首」），下方墊「半透明黑色遮罩 (Dark Overlay)」，中間出現「3D 筊杯」。
    - 使用者點擊擲筊確認這支籤（確認 `confirmStickId`）。
    - 判斷結果：
        - 聖杯 -> 播放籤隻放回動畫，進入【第五關：尋找籤詩抽屜】。
        - 笑杯 / 陰杯 -> 將本次籤號加入「排除清單 (Exclusion List)」， 播放籤隻放回動畫，退回【第三關：抽籤】重抽。（確保亂數池不再抽出已被否決的籤）
6. **第五關：尋找籤詩抽屜**
    - 播放籤隻放回動畫，放回一支「3D 籤」。
        - 「3D 籤筒」放大移至中間。
        - 籤筒右側顯示「第 N 首」提示使用者。
        - 籤筒左邊顯示「點住拖移旋轉籤筒」文字。
    - 對應 `stickId` 的目標抽屜會「亮起閃爍」並帶有「微光外框 (Glow Stroke)」提示。
    - **防呆導航**：若使用者拖曳旋轉超過 5 秒未點擊，系統自動觸發「平滑微調角度 (Auto-snap)」，把目標抽屜緩慢轉向畫面正中央。
    - 使用者拖移籤筒找尋，點擊該目標抽屜。
    - 進入【第六關：閱讀籤詩與解籤】。
7. **第六關：閱讀籤詩與解籤**
    - 播放卡面取出動畫，取出一張「對應卡面」。
        - 中央「3D籤筒」移回原位，上方變成「結束求籤」、「再問一題」熱點。
        - 卡面會迷你狀態從抽屜中拋物線跳出，逐漸放大到中心位置。
        - 卡面下方顯示「儲存圖片」、「放大縮小」按鈕。
    - 顯示詩籤內容與解籤說明。
    - 系統背景儲存紀錄（更新 `isDivination` 狀態為 true）。
    - 提供「儲存圖片」、「放大縮小」按鈕。
    - 結束求籤流程。
